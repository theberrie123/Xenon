FD number       Name    Description
0               stdin   Standard inpute (keyboard)
1               stdout  Standard output (screen)
2               stderr  Standard error (screen, unbuffered)


`write(1, "Hello\n", 6);`
it writes to the screen because 1 = stdout

it reads from the keyboard because 0 = keyboard
