# Makefile
To compile and make the image:
    ```
    make
    ```
To load it in Lithium (node)
    ```
    mv vmxez ../Lithium && cd ../Lithium && make clean && make run
    ```

# Git

To fetch the repo and replace everything:
    ```
    git fetch origin && git reset --hard origin/main && git clean -fd
    ```
